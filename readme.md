# Presetmd

# Usage

```
> git clone https://github.com/Phate334/presentmd.git
> cd presentmd
> git submodule init
> git submodule update
> pipenv install
```

## Run with dev server

* [Flask environment variable](http://flask.pocoo.org/docs/1.0/quickstart/)

create `.env` for pipenv

```
FLASK_APP=web.py
```

* run 

```
> pipenv run flask run
```
